# US_Aluminum_DMFA
This repository contaions the source code and files for the journal article titled "[Decarbonizing the U.S. Aluminum Ecosystem using a Dynamic Material Flow Simulation Framework](https://doi.org/10.1016/j.jclepro.2025.147413)." This work was supported by the [Climate Imperative Foundation](https://climateimperative.org/) and published in the Journal of Cleaner Production in December 2025.

The complete DMFA results covering 1967–2050 are publicly available through the University of Michigan’s [Deep Blue Data repository](https://doi.org/10.7302/ztkj-y672).
